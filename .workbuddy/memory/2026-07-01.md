# 2026-07-01

## 卡牌拖拽系统修复
- **坐标系统修正**：`_start_drag` 改用 `global_position` 计算偏移，`_update_drag` 将全局坐标转为父节点本地坐标
- 添加 `set_process_input(true)` 确保 `_input()` 接收全局事件
- **RiteSlotDrop 重写**：卡槽改为卡牌大小 (88×130)，拖入后显示完整卡面克隆（无头像框）
- **仪式弹窗**：槽位改为横向 HBoxContainer 排列
- **拖放检测**：改用 `set_meta("slot_nodes")` 存储槽位引用，绕过节点路径嵌套问题
- 拖入后原卡牌消失；取消/关闭时恢复；确认时永久移除

## 界面重排（参考原版截图）
- 整体布局改为：顶部状态栏 + 左侧地图 + 右侧固定事件详情面板 + 底部手牌
- 地图节点改为 Control 手动定位，更接近原版路径节点感
- 右侧事件面板替代原先的覆盖弹窗，点击地图仪式即填充详情
- 底部手牌：左侧「俺寻思」骷髅卡、右侧「下一天」按钮
- 角色卡/苏丹卡加回头像框，更接近原版卡牌视觉
